How to run this project on your owm machine (after you clone or download this repository).

1. Check if you have Node.js and npm installed on your computer:
        node -v
        npm -v

2. If you dont't have one, install Node.js and npm for your OS from https://nodejs.org/
When you install Node.js, you automatically get npm installed.

3. Install JSON server:
        npm install -g json-server

4. Go to the project folder and install dependencies:
        npm i

5. Run the project (require ports 3000 and 3004):
        npm run dev
